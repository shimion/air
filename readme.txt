== Description ==

= Avartan Slider has elements Animations, a Drag and Drop Builder, Responsive and Easy to Use =
<br />
Avartan slider also involves other important slider features such as setup of background color or image. 3 kinds of accessible element layers [text, image, and video]
<br />
<br />
<a href="http://demo.solwininfotech.com/wordpress/avartan/">Official WebSite (with live demo)</a>
<br />
= User Friendly Admin Panel =
It has a beautiful and easy-to-use admin interface with modern features like drag and drop slider builder, live preview, and different type of element options.

= High Cross Browser Compatibility =
Avartan Slider is known to be cross browser compatible as it operates superbly on main browsers including Firefox, Opera, Chrome, Safari, and IE.

= Drag and Drop Builder =
How can you position all the elements in to the slider area? You just simply drag them in to the desired position. Like in Powerpoint, you just have to move the object around the Slide.

= Transitions & Animations =
Foreach element and slide you can choose an in animation and an out animation using a simple selection menu. You can set the transition speeds and how long the element will take to get in and out.

= Completely Responsive =
Responsive means that the Slider will be displayed correctly in every resolution that the user will use. If the display is small (like in a smartphone), the slides and the elements will be scaled to be adapted.

= Full & Fixed Width Modes =
Using Avartan Slider you can select between a fixed or a full-width layout (both of them can be responsive). You are the designer, you own the WebSite, just choose the best for it.

== Installation ==

Purchase this plugin directly from http://www.solwininfotech.com/product/wordpress-plugins/avartan-slider/
<br />
1. Go to Plugins > Add New.
<br />
2. Click on Upload Plugin and browse and select avartanslider.zip then click on Install Now.
<br />
3. To activate Avartan Slider, Click on Activate link.
<br />
4. After plugin activation, move on Avartan Slider plugin page.
